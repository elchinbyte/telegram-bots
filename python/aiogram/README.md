# Aiogram Telegram Bots

Collection of Telegram bots built with Python (Aiogram) for learning and experimentation.

## Setup

1. Create virtual environment: `python -m venv .venv`
2. Activate it:  
   - Windows: `.venv\Scripts\activate`  
   - Mac/Linux: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt` (for shared root venv)
